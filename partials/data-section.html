📊 計算公式說明

  • • 最高責任額 = MAX(所有商品責任額)
  • • 最低責任額 = MIN(所有非零責任額)
  • • 最高/最低倍數 = 最高責任額 / 最低責任額
  • • 平均募集天數 = SUM(募集天數) / 商品數
  • • 每日責任額 = 責任額 / 募集天數
  • • 每小時配額 = 每日責任額 / 8
  • • 地獄月份計算：
    1. 依年月分組統計商品數
    2. 找出單月商品數最多的月份
    3. 顯示年份及該月商品數量
